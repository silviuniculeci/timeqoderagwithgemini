# Operator: DayOfWeek
---
### **Description**
Returns the day of week for requested date
---
### **Syntax**
```
: DayOfWeek(<date>, [<culture code - if nothing is provided it uses the current culture>])
ex1. DayOfWeek(Date)
ex1. DayOfWeek(Date, "en-US")
```
